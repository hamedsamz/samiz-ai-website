# راه‌اندازی ایمیل تأیید ثبت‌نام

## بارگذاری

محتویات این پوشه را در ریشه مخزن GitHub بارگذاری کنید و اجازه دهید Vercel آن را Deploy کند.

## متغیرهای لازم در Vercel

- `RESEND_API_KEY`: کلید محرمانه Resend
- `RESEND_FROM_EMAIL`: مقدار `Samiz AI Academy <academy@updates.samizai.com>`

هر دو متغیر باید برای Production فعال باشند. کلید API را داخل GitHub یا کد قرار ندهید.

## روش امن ارسال

1. وارد `/admin/registrations` شوید.
2. از بخش «ارسال آزمایشی» ابتدا فقط به ایمیل خودتان پیام بفرستید.
3. متن، نام فرستنده و پوشه Spam را بررسی کنید.
4. سپس «ارسال به همه افراد ارسال‌نشده» را بزنید.
5. برای افراد خارج از سایت، نام و ایمیل را در فرم جداگانه وارد کنید.

سامانه زمان ارسال را ثبت می‌کند و از ارسال تکراری به ثبت‌نام‌های سایت یا ایمیل‌های دستی جلوگیری می‌کند.
